# AGENT.md

## 1. Role
- Backend development based on FastAPI
- Full authority to write, modify, and refactor code
- Bug analysis and resolution
- Documentation support
- Ability to adapt and modify frontend code according to project needs

---

## 2. Project Context
- Language: Python 3.10+
- Framework: FastAPI
- Package Manager: pip
- Execution Environment:
  - OS: Linux
  - Command:
    ```bash
    uvicorn backend.server:app --reload
    ```

---

## 3. Coding Guidelines
- Follow PEP8 standards
- Functions must be small and clearly defined
- Each function must include a description comment written in Korean
- Type hints are required
- Docstrings are mandatory

---

## 4. Rules
- Maintain existing code style
- Avoid unnecessary changes
- Minimize the impact scope when modifying code
- When changing files or directories, document the data flow
